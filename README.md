# 局域网工具箱 v2.1

手机浏览器打开即可用，无需安装 App。

## 功能

- **📦 文件传输** — 拖拽上传、一键下载、删除管理，支持大文件流式传输不爆内存
- **🎮 手机遥控** — 触摸板控制鼠标 + 按住拖动 + 键盘输入 + 快捷键

### 遥控器

```
┌───────────────────┬──────┐
│                   │ 左键 │
│     触摸板         ├──────┤
│  单指=鼠标 双指=滚动│ 右键 │
└───────────────────┴──────┘
🎯精准 🐢慢 🐇标准 🚀快 ⚡疯狂
⏮ ⏯ ⏭ 🔉 🔇 🔊
```

- 🖱️ **鼠标加速度** — 慢滑精准，快甩飞越屏幕
- ✌️ **双指滚动** — 不用切换模式
- 🔽 **按住拖动** — 左键/右键按下不松，配合触摸板拖拽窗口
- ⚡ **125Hz 刷新率** — 8ms 间隔，接近游戏鼠标手感
- ⚙️ **5 档灵敏度** — 一键切换，自动保存
- 📺 **全屏模式** — 隐藏浏览器 UI
- 💡 **屏幕常亮** — 遥控时不熄屏
- 🎵 **媒体控制** — 播放/暂停、上下曲、音量

### 快捷键

Tab / Esc / 退格 / 空格 / Enter / 方向键 / Win / Alt+Tab / 任务视图

## 使用

```bash
PYTHONIOENCODING=utf-8 python dropbox.py 8888
```

手机和电脑连同一个 WiFi，手机浏览器打开 `http://电脑IP:8888` 即可。

## 技术

- 纯 Python，零外部依赖
- 鼠标/键盘通过 Win32 API（SendInput）实现
- `ThreadingHTTPServer` 多线程处理
- HTTP keep-alive 连接复用
- 文件上传/下载 1MB 分块流式传输，不爆内存
- 触摸板 8ms 间隔 + 加速度曲线 + 请求去重
- localStorage 持久化灵敏度设置
